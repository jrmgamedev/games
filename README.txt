Cats And Boxes

A casual game inspired by the classic favorite board game — Snake and Ladders.

Features:
• Turn-Based Gameplay
• 8 Awesome Cats
• 6 Unique Maps
• Local Multiplayer

Note: Local Multiplayer can be played by 2 (up to 8) people by sharing a single device.

Maps:
• Tiny Island
• Rocky Mountain
• Spooky Garden
• Ocean Park
• Classic Tower
• Game World

Supported Languages:
• English
• Español
• 日本語
• 한국어
